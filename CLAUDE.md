# Global shared files

Read these at the start of any relevant task:
- `WRITING-PREFERENCES.md`
- `RTF-FRAMEWORK.md`

Apply `WRITING-PREFERENCES.md` to any writing, rewriting, summarisation, documentation, issue text, or user-facing prose.

Apply `RTF-FRAMEWORK.md` when handling unstructured prompts. Check Role, Task, and Format. If any element is missing or ambiguous, ask concise clarification questions. If the user says "just run it" or "skip questions", execute immediately.

Skip RTF evaluation for structured workflows: Engineering prompts, Prototypes prompts, and Developer feedback workflow.

If local instructions conflict with these global files, ask which source takes precedence.
